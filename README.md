# 一键刷新网络（Windows 免重启）

一个单文件 `.bat` 脚本，双击即可刷新 Windows 网络配置，**全程不需要重启电脑**。

## 功能

脚本依次执行以下操作（均为「不需要重启」的项目）：

| 步骤 | 命令 | 说明 |
|------|------|------|
| 1/3 | `ipconfig /flushdns` | 刷新 DNS 解析缓存 |
| 2/3 | `ipconfig /renew` | 重新获取 IP 地址（DHCP 续租） |
| 3/3 | `ipconfig /registerdns` | 重新注册 DNS 名称（**仅管理员时执行**，非管理员自动跳过） |
| 自检 | `ping 8.8.8.8` / `ping www.baidu.com` | 测试网络连通性 |

每一步都会在弹出的黑窗口里显示**真实输出 + 中文标注**（成功 / 跳过均有提示），让你明确知道脚本确实跑过了。

> **为什么"免重启"**：Windows 里还有 `netsh winsock reset` 和 `netsh int ip reset` 两种重置方式，但它们执行后**必须重启电脑**才生效。本脚本按需求刻意排除了这两步，只做即时生效的刷新。

## 使用方法

1. **双击 `一键刷新网络.bat`** 即可运行，黑窗口会一步步显示刷新过程，最后「按任意键退出」保持窗口方便你查看结果。
2. 想让第 3 步 `registerdns`（DNS 重新注册）也生效：**右键脚本 → 以管理员身份运行**。

## 兼容性 / 实现说明

- 适用：Windows 10 / 11。
- 脚本以 **UTF-8 带 BOM** 保存，并在首行执行 `chcp 65001`，因此中文能稳定显示，不会出现乱码或 `'xxx' is not recognized` 报错。
- 不依赖 Python 或任何外部程序，纯 Windows 原生命令。

## 适用场景

- 网页打不开 / DNS 解析异常，想快速清缓存。
- 换了网络环境（插拔网线、切换 Wi-Fi）后 IP 没更新。
- 日常维护、排查网络前的"先刷新一遍"。
